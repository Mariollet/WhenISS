Politique de Confidentialité - WhenISS

Cette politique de confidentialité décrit comment votre information est utilisée et protégée dans notre application.

1. Déclaration sur la collecte de données

Nous confirmons que notre application ne collecte aucune donnée utilisateur. Cela inclut, mais sans s'y limiter :

  • Données personnelles (nom, adresse, email, numéro de téléphone).
  • Données de localisation.
  • Données financières.
  • Informations sur les appareils.

2. Partage de données

Aucun partage de données utilisateur n'est effectué avec des tiers. Votre confidentialité est notre priorité.

3. Utilisation de bibliothèques et SDK

L'application n'utilise pas de bibliothèques ou de SDK qui transmettent des données utilisateur, directement ou indirectement.

4. Technologies WebView

Si notre application utilise une fonctionnalité WebView, elle ne collecte ni ne transfère de données utilisateur lors de la navigation.

5. Traitement éphémère des données

Si notre application traite des données de manière éphémère (par exemple, pour une réponse en temps réel), ces données ne sont jamais stockées ou partagées.

6. Modifications de cette politique

Nous nous réservons le droit de mettre à jour cette politique de confidentialité si nécessaire. Toute modification sera clairement indiquée dans cette page.

7. Contact

Pour toute question concernant cette politique de confidentialité, veuillez nous contacter à : 8amos8@gmail.com.
